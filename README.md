# Calculadora

Projeto de uma calculadora em C# que realiza operações matemáticas básicas e mantém um histórico das operações.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Potência
- Raiz quadrada
- Histórico de operações
- Sair do programa

## Como usar
1. Abra o projeto no Visual Studio.
2. Compile e execute.
3. Siga as instruções no menu.

## Tecnologias
- Linguagem: C#
- Plataforma: .NET
